# 更新日志

本项目遵循“未发布变更优先记录”的方式维护更新说明。金额均为 API 等价估算，不代表 ChatGPT/Codex 订阅账单或实际扣款。

## Unreleased — 2026-07-21

### 新增

- 新增环形额度视图，并保留卡片视图切换。
- 新增 Token 消耗面板，分别显示普通输入、缓存读取、可见输出与推理输出。
- 新增当日与累计统计；累计周期支持近 7/30/90 天、全部及自定义日期。
- 新增 Standard API 等价金额和独立的 Standard、Priority/Fast、Flex 档位金额估算。
- 新增速度档位金额全 Token 覆盖：缺失档位按 Standard 推定，内部或无公开价格模型使用当前公开配置模型的同档位参考价。
- 新增直接定价与兜底推定 Token 比例显示，并在提示中列出参考模型和触发原因。

### 改进

- 窗口默认尺寸缩小约 50%，支持四边及四角拉伸。
- 界面采用等比缩放，避免非等比例拉伸时文字、圆环和图标变形。
- 增加窗口宽高、透明度、强调色、背景色、置顶状态及 Token 显示选项的持久化设置。
- 按逐模型官方价表计算不同速度档位，不再使用统一倍率近似；Codex `fast` 映射为 Priority。
- 保留原 Standard 等价价格口径，新的速度档位金额不会覆盖原金额。
- 精简 README，使其保持为安装与使用入口；详细定价和兜底规则移至独立说明文档。

### 数据与隐私

- Token 统计仅从本地会话日志提取计数、模型、档位、时间戳和会话边界，不读取或上传提示词与回复正文。
- 仅为内部模型选择公开参考价而读取 `~\.codex\config.toml` 顶层 `model` 字段，不读取凭据或密钥。
- 完善仓库忽略规则、隐私说明和提交前脱敏检查。

- 运行版已验证显示“全覆盖推算”及推定比例。
